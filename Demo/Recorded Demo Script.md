# 90-Second Recorded Demo Script

## Visual Flow + Narration

**[0-15s] Hook + Problem**
*"Traditional peptide design takes years of trial and error. We've built an AI system that reduces this to weeks through intelligent iteration."*

**[15-45s] Show Pipeline Flow** 
*[Point to flowchart] "Here's how it works: We start by generating 1000 peptide candidates using BoltzGen."*

*[Highlight SCORE] "Then we score them with three different AI models - Boltz-2 for binding, Vina for physics, GNINA for machine learning."*

*[Highlight ANALYZE] "Next, we run integrated gradients on our top candidates using Chai-2, Boltz-2, and ESM-2 to identify which amino acid positions are critical."*

*[Highlight LOCK] "When all models agree a position is important, we lock it. Then we generate the next batch with these constraints."*

**[45-75s] Show Results**
*[Point to progress visualization] "Each iteration gets consistently better. Our progress bar shows the steady improvement from initial candidates to fully optimized peptides."*

*"Instead of random changes, we're making surgical optimizations that preserve what works while enhancing binding performance."*

**[75-90s] Close**
*"This is the first platform to combine three AI models for peptide design. We're ready to scale from computational validation to lab testing."*

## Screen Recording Notes

### Timing Cues
- **0-15s**: Static title slide → flowchart reveal
- **15-30s**: Highlight CREATE box, show candidate generation
- **30-45s**: Animate through SCORE → ANALYZE → LOCK sequence  
- **45-60s**: Zoom to results section, highlight progression
- **60-75s**: Show iteration arrow, emphasize the cycle
- **75-90s**: Pull back to full pipeline, end on results

### Visual Effects
- **Box highlighting**: Glow effect on current step
- **Arrow animation**: Trace the flow path as you explain
- **Results reveal**: Animate the generation boxes left-to-right
- **Iteration emphasis**: Pulse the iteration counter

### Optional Data Overlays
- Show actual peptide sequences for visual interest
- Display real IG heatmaps during ANALYZE section
- Animate binding affinity scores counting up

## Technical Setup

**Video Specs:**
- 1920x1080 resolution minimum
- Clean background (white/light gray)
- Professional font rendering
- Smooth transitions between sections

**Audio:**
- Clear, confident delivery
- Pause at commas for emphasis
- Speed up technical details, slow down on results
- End with conviction on "ready to scale"
